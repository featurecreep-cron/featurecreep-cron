# feature_creep

It was supposed to be one container.

Now there are thirty and I keep writing about what went wrong.

## Projects

- **[Morsl](https://github.com/featurecreep-cron/morsl)** — generate weekly menus from your Tandoor Recipes collection. Constraint-based recipe selection, household ordering, meal plan sync. Docker image on GHCR.
- **[Roustabout](https://github.com/featurecreep-cron/roustabout)** — Docker environment documentation generator. Reads from the Docker socket, produces structured markdown with automatic secret redaction.
- **[tandoor-client](https://pypi.org/project/tandoor-client/)** — auto-generated Python client for the Tandoor Recipes API. Published to PyPI daily when the upstream API changes.
